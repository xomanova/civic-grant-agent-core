# Civic Grant Agent Core

Welcome to the Civic Grant Agent Core documentation! This project aims to automate and streamline the grant application process for civic organizations, particularly fire departments and similar public service entities.

## What is Civic Grant Agent?

Civic Grant Agent is an intelligent multi-agent system designed to help civic organizations discover, evaluate, and apply for grants. The system uses AI-powered agents to automate time-consuming tasks in the grant application process.

## Key Features

- **Automated Grant Discovery**: Intelligent web search to find relevant grant opportunities
- **Eligibility Validation**: Automated analysis of grant requirements against department profiles
- **Grant Writing Assistance**: AI-powered draft generation for grant applications
- **Multi-Agent Architecture**: Specialized agents working together for optimal results

## System Overview

The Civic Grant Agent Core operates through three specialized agents:

1. **GrantScout** - Searches and identifies potential grant opportunities
2. **GrantValidator** - Analyzes and validates grant eligibility
3. **GrantWriter** - Generates grant application drafts

Each agent is powered by advanced language models and uses specialized tools to accomplish its tasks.

## Use Cases

- **Fire Departments**: Finding equipment and training grants
- **Volunteer Organizations**: Discovering funding opportunities
- **Municipal Agencies**: Streamlining grant application processes
- **Non-Profit Organizations**: Automating grant research and writing
